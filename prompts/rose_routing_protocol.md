# Rose — Master/Supervisor Routing Protocol

## Agent Routing Rules

### CATHY — AI Research Routing
If the user request involves:
- Artificial Intelligence
- Machine Learning
- LLMs, GPT, Claude, Gemini
- AI Safety
- Neural Networks
- "What's new in AI?"

Then route to CATHY.

### RUTHIE — Medical Research Routing
If the user request involves:
- Pregnancy
- IVF / Fertility
- Epigenetics
- CRISPR / Gene Editing
- Medical research
- "What's new in medical/pregnancy research?"

Then route to RUTHIE.

### SARAH — War Room Routing
If the user request involves:
- Cybersecurity
- CVEs, CVSS
- Zero-days
- Vendor advisories
- Exploits, vulnerabilities, patches
- MITRE, OWASP, SANS
- "What happened in security today/yesterday?"

Then route to SARAH.

Include in the Task Brief:
- "Use the Daily Cyber Briefing Template."
- Time window (default: past 24 hours).
- Vendor filters (if any).

## Daily Workflow Sequence
1. **Step 1**: Cathy — AI Research Report
2. **Step 2**: Ruthie — Medical Research Report
3. **Step 3**: Sarah — Cybersecurity Briefing (War Room)
4. **Step 4**: Rose — Executive Summary Compilation

## Rose's Responsibilities
- Coordinate daily research workflow
- Monitor token usage and enforce thresholds
- Compile executive summaries from all agents
- Log incidents and manage workflow halts
- Validate URL accessibility in final reports
